# Academic Project Tracker

A web-based academic project management system built with Python and Flask. This application helps students track their projects, manage sub-tasks, calculate deadlines, and monitor project budgets efficiently.

## Features

* **User Authentication:** Secure registration and login system with data isolation (users only see their own projects).
* **[US1] Project & Deadline Management:** Create projects and automatically calculate the remaining days or overdue status based on deadlines.
* **[US2] Task Tracking:** Add sub-tasks to projects, mark them as completed/pending, and view real-time overall progress percentages.
* **[US3] Budget Management:** Define a project budget, add expenses, delete incorrect expenses, and automatically track the remaining budget with visual warnings for overspending.
* **Raw SQL Implementation:** Fully utilizes raw SQL queries without ORM frameworks for database operations.

## Technologies Used
* **Backend:** Python, Flask
* **Database:** SQLite3 (Raw SQL)
* **Frontend:** HTML5, CSS3
